# Radha Langoju — Portfolio

A fully static, single-page portfolio. Plain HTML + CSS + vanilla JavaScript — no build step, no config.

## Files

- `index.html` — the site
- `styles.css` — styles (light/dark themes)
- `script.js` — theme toggle, scroll reveals, stat count-up
- `Radha_Langoju_Resume.pdf` — résumé download

**Keep all files in the same folder** so the résumé download and asset links resolve.

## Deploy

### Netlify
Drag-and-drop this folder into the Netlify dashboard (Sites → "Add new site" → "Deploy manually"), or connect the GitHub repo:
- Build command: **none**
- Publish directory: **root** (`/`)

### GitHub Pages
1. Push this folder to a repo (e.g. `lradha2812-cmyk/personal-website`).
2. Repo → Settings → Pages → Source: **Deploy from a branch** → `main` / `/ (root)`.
3. The site publishes at `https://<username>.github.io/<repo>/`.
